Expert talk on Travelling Salesman Problem by Mr. Lokesh Jain

ACM Student Chapter IGDTUW organized an Expert Talk on Optimization Techniques for Social Good by Mr. Lokesh Jain under Summer Workshop cum Internship 2021 on 18th June,2021 from 12pm to 2pm.In this session, attendees were given a problem related to 'Traveling Salesman' to be solved using ACO: Ant Colony Optimization algorithm. A thorough analysis of the code was done at the end of the session. Talk was lauded by 50+ trainees.
